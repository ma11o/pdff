## pdff

### Usage

```
Usage: pdff [options] [command]

Options:
  -h, --help                     display help for command

Commands:
  compress|cp [file]             Compress pdf file
  split|s [file] [start] [last]  Split pdf file
  combine|cb [file1] [file2]     Combine pdf file
  help [command]                 display help for command
```
